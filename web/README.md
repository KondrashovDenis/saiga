# Saiga Web App

Веб-приложение для взаимодействия с русскоязычной языковой моделью Saiga на базе Llama 2.

## Возможности

- Пользовательские аккаунты и аутентификация
- Создание и сохранение диалогов с моделью
- Настройка параметров генерации текста
- Возможность делиться диалогами по ссылке
- Темный и светлый режимы интерфейса

## Структура проекта

Проект включает в себя:
- Бэкенд на Flask (Python 3.11)
- База данных SQLite через SQLAlchemy
- Аутентификация на Flask-Login
- Интерфейс на Bootstrap 5 и JavaScript
- Контейнеризация через Docker
- Обратный прокси на Nginx

## Установка и запуск

1. Настройте переменные окружения в docker-compose.yml
2. Запустите приложение через Docker Compose
3. Настройте Nginx для проксирования запросов
4. Доступ к приложению по адресу: https://saiga.denciaopin.com

## Интеграция с LLM API

Приложение интегрируется с API языковой модели через конфигурируемый URL.
Предполагается совместимость с форматом text-generation-webui.
